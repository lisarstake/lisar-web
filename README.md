# Lisar

A platform that makes crypto staking simple and accessible to everyone. Earn up to 68% APY by staking on blockchain networks using your local currency - no crypto knowledge required.

## About Lisar

Lisar is a user-friendly platform that lets anyone earn passive income through blockchain staking. We handle all the complex technical aspects behind the scenes, so you can focus on earning rewards.

**Key Features:**
- **Stake with Fiat** - Use your regular money, no crypto needed
- **High Returns** - Earn up to 68% APY
- **Multiple Networks** - Currently Livepeer, with BNB and Solana coming soon
- **Simple Interface** - Easy to use, no technical knowledge required
- **Flexible Withdrawals** - Access your funds anytime

## Project Structure

This is a monorepo containing:

- **`apps/web`** - Public-facing web application for users
- **`apps/admin`** - Administrative dashboard for platform management

## Requirements

- Node.js 18 or higher
- npm or pnpm
